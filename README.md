# Testlo API

1. Clonar project
2. ```npm i```
3. Clonar el archivo ```.env.template``` y renombralo a ```.env```
4. Cambiar las variable de entorno
5. Levantar la base de datos
  ```
  docker-compose up -d
 ```
6. Levantar: ```npm run start:dev```
